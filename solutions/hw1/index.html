		Homework 1: 		Function Prototypes,

Machine Constants,

and  Critical Values

William J. De Meo

October 7, 1997

Abstract:

This paper presents programs which exhibit three fundamental programming topics. First we consider how function prototyping helps to guard against passing variables of different data types to a function. Then we evaluate machine constants, such as the largest value of each data type and the smallest floating point number in single and double precision. Finally, we construct a program which supplies the critical value corresponding to user specified degrees of freedom and probability level.
